# Rock Paper Scissors Game

A simple command-line Rock Paper Scissors game built in Python.

## How to Play

Run the game and follow the prompts to choose rock, paper, or scissors.

The computer will make a random choice, and the winner is determined by the rules:

- Rock beats scissors
- Scissors beat paper
- Paper beats rock

## Features

- Input validation for user choices
- Random computer selection
- Score tracking across multiple rounds
- Option to play again
- User-friendly interface with clear instructions and feedback

## Requirements

- Python 3.x

## Running the Game

1. Ensure you have Python installed.
2. Open a terminal in the project directory.
3. Run the command: `python game.py`

## Game Rules

- Rock beats scissors
- Scissors beat paper
- Paper beats rock
- If both choose the same, it's a tie

Enjoy the game!
